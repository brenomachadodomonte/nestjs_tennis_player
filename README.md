## Description

An API to Ranking Tennis Players

## Mongo Installation

```bash
$ docker run -p 27017:27017 -d --name mongodb mongo
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
